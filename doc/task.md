# Задача: Разработка алгоритма упрощения KML-областей

## Основное задание:

1. Разработать алгоритм упрощения (минимизации кол-ва вершин) областей из файла KML.

### Входные данные:
- Область с избыточным количеством точек
- Максимально допустимое отклонение границ упрощенной области от исходной в метрах

### Результат:
- Упрощённая область

**Задача:** получить область с минимально возможным количеством точек, не превышая допустимое отклонение границ упрощенных областей от исходных.

## Дополнительные требования (будет плюсом):

- Область будет загружаться из файла KML (предоставим)
- Реализовать графический интерфейс пользователя для выбора исходного файла KML, индикации процесса упрощения и вывода результатов на экран
- Реализовать выполнение алгоритма в отдельном потоке для сохранения работоспособности графического интерфейса приложения во время работы алгоритма
- После прочтения областей из KML вывести их на экран (исходные области черным контуром, упрощенные – зелёным) с возможностью масштабирования и сдвига области отображения мышью для просмотра областей на любом масштабе (вывод областей на тайловую спутниковую карту будет плюсом)
- Сохранить все атрибуты областей кроме координат
