# Алгоритм упрощения полигонов: Ramer–Douglas–Peucker (RDP)

## Описание
Рекурсивный алгоритм упрощения ломаной (полигона), сохраняющий её основную форму.  
Удаляет точки, которые находятся ближе, чем заданное расстояние `epsilon`, к отрезку между двумя опорными точками.

## Алгоритм

0. **Сигнатура функции**:

      ```QPolygonF Model::simplifyPolygon(QPolygonF polygon, double epsilon) ```

1. **Вход**:  
   - Полигон как последовательность точек `polygon = [P_0, P_1, ..., P_n_minus_1]`,
   где координаты точек полигона подаются в метрической системе;
   - Пороговое расстояние `epsilon > 0`.

2. **Базовый случай**:  
   Если в полигоне ≤ 2 точек — возвратить его как есть.

3. **Основной шаг**:  
   - Пусть `A = P_0` (первая точка), `B = P_n_minus_1` (последняя точка).  
   - Построить отрезок `AB`.  
   - Для всех промежуточных точек `P_i` (`i = 1 ... n−2`) вычислить **перпендикулярное расстояние** до отрезка `AB`.  
   - Найти точку `C = P_k` с **максимальным расстоянием** `d_max`.

4. **Решение**:  
   - Если `d_max <= epsilon`:  
     -> Все промежуточные точки **отбрасываются**. Результат: `[A, B]`.  
   - Если `d_max > epsilon`:  
     -> Точка `C` **сохраняется**.  
     -> Рекурсивно применить алгоритм к подполигонам:  
       - `[A, ..., C]`  
       - `[C, ..., B]`  
     -> Объединить результаты (без дублирования `C`).

5. **Выход**: упрощённый полигон.

## Сложность алгоритма
  - Худшая: **O(n^2)**  
  - Средняя: **O(n*log*(n))**
